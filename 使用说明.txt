
矩阵计算器 使用说明
=========================================

【一、软件简介】
--------------------------
本软件是一个基于 Python 和 tkinter 开发的图形界面矩阵计算器，支持：

- 多种方式导入或生成矩阵（TXT、CSV、图片识别、随机分布等）
- 支持复杂矩阵运算与分解（LU、QR、SVD等）
- 支持矩阵的可视化展示与导出
- 自动保存计算历史、支持查看与恢复

适用于教学演示、科研辅助、工程计算等场景。

【二、运行方法】
--------------------------
1. **直接运行可执行文件**
   - 打开 `MatrixCalculator.exe`（需先完成打包）
   - 无需命令行操作，图形界面自动弹出

2. **开发调试（Python 环境）**
   - 安装 Python 3.8+
   - 安装所需依赖：
     ```
     pip install -r requirements.txt
     ```
   - 运行 `python test.py`

【三、界面说明】
--------------------------
启动软件后，主界面分为以下部分：

1. **控制面板**
   - 导入矩阵A、矩阵B 或 数值b
   - 选择操作类型并执行计算
   - 导出计算结果
   - 主题切换与历史记录查看

2. **矩阵A 区**
   - 查看、编辑、确认矩阵A
   - 支持可视化、导出

3. **矩阵B/数b 区**
   - 可作为矩阵B或标量b使用
   - 可视化、导出、确认

4. **计算结果区**
   - 显示操作结果或错误提示
   - 支持结果可视化（热图、3D柱状图、特征向量）

【四、使用流程】
--------------------------
1. **导入或输入矩阵**
   - 可通过“导入矩阵A/B”按钮，选择 TXT、CSV 或 JPG 文件
   - JPG 文件会自动识别为矩阵（OCR）
   - 也可手动粘贴矩阵到输入框（支持分数、复数）

2. **确认矩阵**
   - 点击“确定”按钮将输入内容转换为可计算矩阵（sympy）
   - 支持检测输入合法性，自动识别为矩阵或标量

3. **选择操作类型**
   - 从下拉菜单中选择所需操作（共20+种）
   - 如果操作需要矩阵B或数b，请确保其已确认

4. **执行计算**
   - 点击“执行计算”开始运算
   - 结果将在下方“计算结果区”显示
   - 如有错误或维度不匹配，将显示提示

5. **结果处理**
   - 可视化：点击“结果可视化”
   - 导出：点击“导出结果”保存为 TXT 或 CSV

6. **历史记录**
   - 点击“历史记录”可查看最多50条历史
   - 支持查看、导出、加载回主界面

【五、手动输入格式说明】
--------------------------
1. ✅ **支持的数据类型**
   - **整数**：如 `1`, `42`, `-3`
   - **小数**：如 `3.14`, `-0.5`
   - **分数**：如 `1/3`, `-5/7`
   - **复数**（虚数）：如 `2+3i`, `-1.5-0.5i`, `i`, `3i`, `1/2+i/3`  
     👉 **注意：`i` 代表虚数单位**，输入时大小写均可（`i` 或 `I`）

2. ✅ **矩阵输入格式要求**
   - **按行输入，每行一个矩阵行**
   - **同一行内各元素之间以空格分隔**
   - 示例输入：
     ```
     1 2 3
     4 5 6
     7 8 9
     ```

   - 示例带分数与复数：
     ```
     1/2 3 4+2i
     -1 i 5/6
     0 0 -3i
     ```

3. ✅ **作为单个数值输入（用于数b）**
   - 如果在“矩阵B/数b”框中仅输入一个数字（不含换行），程序会自动识别为标量 `b`

4. ⚠️ **不支持的格式示例**
   - 使用英文逗号 `,` 分隔
   - 表格形式粘贴未清理空格
   - 错误的复数格式，如 `3 + i2`（应为 `3+2i`）

【六、操作支持列表】
--------------------------
基础运算：加法A+B、减法A-B、数乘b×A、乘法A×B  
矩阵属性与变换：秩、转置、行列式、逆矩阵、共轭转置、幂运算A^b、范数、迹  
矩阵分析：特征值、特征向量、AX=B求解  
矩阵结构操作：行列交换、拼接（水平/垂直）  
分解操作：LU分解、QR分解、奇异值分解（SVD）

【七、随机矩阵生成】
--------------------------
可基于以下分布生成矩阵：正态、均匀、泊松、指数。  
请设置参数1、参数2，行列数，并选择目标矩阵后点击“生成”。

【八、可视化说明】
--------------------------
- 小矩阵（≤10×10）：3D柱状图，显示具体值  
- 大矩阵：热力图  
- 特征向量：支持 2×2（2D箭头图）与 3×3（3D箭头图）

【九、导出功能】
--------------------------
可将以下内容导出为 `.txt` 或 `.csv`：矩阵A、B、结果、特征值/向量、分解结果、历史记录  
图像可导出为 `.png`, `.jpg`, `.pdf`, `.svg`

【十、注意事项】
--------------------------
- 所有输入建议点击“确定”确认后再计算
- 幂运算仅支持整数，负幂需矩阵可逆
- 方阵限定操作：逆、行列式、特征值/向量、迹、LU/QR分解
- OCR功能首次运行需联网下载模型
- 特征向量可视化仅支持 2×2 和 3×3
- 导出路径需可写权限

【十一、依赖环境】
--------------------------
Python >= 3.8  
依赖模块：numpy, sympy, matplotlib, paddlehub, arrow, shapely, pyclipper, ttkbootstrap, opencv-python  
安装方式：
```
pip install -r requirements.txt
```
